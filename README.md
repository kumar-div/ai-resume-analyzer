# 🚀 AI Resume Analyzer

> Built by **Divyansh**  
> Not another resume tool — this actually tells you *why your resume fails*.

---

## 🧠 What this does

An AI-powered resume analyzer that:

- Scores your resume like an ATS system  
- Detects missing & weak keywords (not just exact match)  
- Identifies real skill gaps  
- Generates actionable improvements  
- Rewrites your resume for better impact  

---

## ⚡ Why this is different

Most tools:
- ❌ Just match keywords  
- ❌ Give generic advice  
- ❌ Don’t improve your resume  

This tool:
- ✅ Uses semantic + fuzzy matching  
- ✅ Highlights real weaknesses  
- ✅ Shows before vs after improvement  
- ✅ Stores your past analyses  
- ✅ Works offline with local AI  

---

## 🧩 Core Features

- 📊 ATS Score with breakdown  
- 🔍 Matched / Missing / Partial keywords  
- 🚨 Critical Issues detection  
- 🤖 AI Resume Rewrite  
- 📄 Resume Comparison (Original vs Improved)  
- 💾 Dashboard (saved analyses)  
- 🔐 Google & GitHub login  

---

## 🛠 Tech Stack

- **Frontend:** Next.js, TypeScript, Tailwind CSS  
- **Backend:** Next.js API Routes  
- **Database:** Prisma + SQLite  
- **Auth:** NextAuth  
- **AI:** Ollama (Local LLM)  

---

## ⚙️ Setup

- git clone https://github.com/kumar-div/ai-resume-analyzer.git
- cd ai-resume-analyzer
- npm install

---

## 🤖 Run local AI (Ollama)

ollama run phi

---

## 🤖 Start app

npm run dev
